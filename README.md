# Penton - Sistema de Gestión de Inventario

Sistema web para gestionar y visualizar el inventario de productos del negocio **Pentón**.

## 📋 Información del Negocio
- **Nombre:** Pentón
- **Localidad:** Matanzas, Cuba
- **Teléfono:** 52620551

## 🚀 Características
- ✅ Login seguro del dueño
- ✅ Panel de administración para subir/editar productos
- ✅ Catálogo público visible para clientes
- ✅ Gestión de inventario (nombre, descripción, precio)
- ✅ Base de datos sin Firebase

## 🛠️ Tecnologías
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Node.js + Express
- **Base de datos:** MongoDB
- **Autenticación:** JWT

## 📁 Estructura del Proyecto
```
penton-inventory/
├── frontend/          # Archivos HTML, CSS, JS del cliente
├── backend/           # API Node.js + Express
├── database/          # Configuración de MongoDB
└── README.md
```

## 🔧 Instalación

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
Abre `frontend/index.html` en el navegador
